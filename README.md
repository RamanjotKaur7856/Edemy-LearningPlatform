# Edemy – Student Learning Platform

## 🚀 Overview
Edemy is a full-stack web application designed to manage users, courses, enrollments, and quizzes. It provides real-time quiz functionality for interactive learning.

## 🛠 Tech Stack
- Frontend: React  
- Backend: Node.js, Express  
- Database: MongoDB  
- Real-time: WebSockets  
- Deployment: Docker  

## ✨ Features
- User authentication and authorization  
- Course and enrollment management  
- Real-time quiz system  
- RESTful APIs for backend services  
- Input validation and error handling

- Designing REST APIs  
- Real-time communication using WebSockets  
- Full-stack development workflow
